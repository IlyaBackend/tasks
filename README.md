# Tasks

Практические задачи на Python с соблюдением PEP8.

## Запуск

1. Установите Python >= 3.10
2. Установите uv: https://github.com/astral-sh/uv
3. Установите зависимости:
   uv sync
4. Проверка и форматирование кода:
   uv run ruff format .
   uv run ruff check .

Весь исходный код находится в директории `src/`.
